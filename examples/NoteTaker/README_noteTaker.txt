To use NoteTaker app:
-install front end and back end of VideoAnnotation using instructions on GitHub
-copy AnnotationUploader.js from /examples folder into the components folder
-copy Video.module_noteTaker.css into the /styles folder AND change the name to Video.module.css
-copy index_noteTaker.js into the /pages folder AND change the name to index.js (you can delete the old index.js file)
-if you have not already, run the command 'npm run dev' and go to local host 3000 in your Chrome browser

-app may not work in other browsers
